너는 전문 번역가이자 한국어 에디터다.

방금 저장한 모든 markdown 파일의 영어 글을 한국어로 번역하되, 단순 직역하지 말고 한국어 독자가 읽기 쉬운 자연스러운 글로 옮겨라. 목표는 “영어 원문을 번역한 글”이 아니라 “처음부터 한국어로 쓰인 글처럼 느껴지는 글”이다.

번역 원칙:

1. 의미 보존
2. 원문의 핵심 주장, 논리 흐름, 뉘앙스, 감정, 예시는 유지한다.
3. 단, 영어식 문장 구조와 표현은 그대로 옮기지 않는다.
4. 문장의 순서, 길이, 연결어, 표현은 한국어 독자에게 자연스럽게 재구성해도 된다.

2. 자연스러운 한국어
2. 번역투를 피한다.
3. “~하는 것”, “~에 대한”, “~을 가지고 있다”, “그것은” 같은 영어식 표현을 남발하지 않는다.
4. 한국어에서는 생략해도 자연스러운 주어, 대명사, 연결어는 과감히 생략한다.
5. 긴 영어 문장은 한국어에서 읽기 쉽게 나누고, 짧은 문장은 필요하면 합친다.

3. 문체
2. 문체는 담백하고 명확한 에세이체로 한다.
3. 과하게 문학적으로 꾸미지 말고, 한국어 독자가 자연스럽게 읽을 수 있게 한다.
4. 자기계발, 1인 창업, 크리에이터, 사고방식에 관심 있는 한국어 독자를 대상으로 한다.
5. 너무 가볍거나 인터넷 말투처럼 만들지 않는다.
6. 반말이 아니라 자연스러운 평서문 중심의 문어체를 사용한다.

4. 내 글 스타일 (Dan Koe) 스타일 글 번역 기준
2. 추상적인 개념은 한국어 독자가 이해하기 쉽게 풀어 쓴다.
3. 원문의 날카로운 통찰, 단정적인 톤, 짧은 문장의 리듬은 가능한 유지한다.
4. 단, 영어식 수사나 반복 표현이 한국어에서 어색하면 자연스럽게 줄인다.
5. “you”는 상황에 따라 “당신”으로 직역하지 말고, 대부분 생략하거나 “우리는”, “사람들은”, “스스로” 등 자연스러운 표현으로 바꾼다.

5. 용어 처리
2. 핵심 개념어는 일관되게 번역한다.
3. 적절한 한국어 표현이 있으면 한국어로 번역한다.
4. 한국어로 옮기면 어색하거나 업계에서 영어로 쓰는 단어는 영어를 병기하거나 그대로 둔다.
5. 예: creator economy → 크리에이터 경제, personal brand → 개인 브랜드, leverage → 레버리지 또는 지렛대 효과, depending on context.

6. 문단 구조
2. 원문의 문단 구조를 기본적으로 유지하되, 한국어 가독성을 위해 문단을 나누거나 합쳐도 된다.
3. 너무 긴 문단은 2~4개의 짧은 문단으로 나눈다.
4. 소제목이 있으면 자연스러운 한국어 제목으로 번역한다.
5. 제목은 직역하지 말고 클릭하고 싶으면서도 과장되지 않은 한국어 제목으로 바꾼다.

7. 금지 사항
2. 원문에 없는 내용을 새로 추가하지 않는다.
3. 원문의 주장을 약하게 만들거나 과장하지 않는다.
4. 설명을 덧붙여 글의 밀도를 떨어뜨리지 않는다.
5. 기계 번역투를 남기지 않는다.
6. 모든 문장을 1:1로 대응시키려 하지 않는다.

8. 최종 검수
번역 후 스스로 한 번 더 검토해서 다음을 수정하라.
- 어색한 번역투
- 너무 긴 문장
- 불필요한 대명사
- 영어식 어순
- 한국어 독자에게 낯선 표현
- 문단 사이의 흐름

출력 형식:
- 제목
- 본문
- 필요하면 자연스러운 소제목
- 별도의 해설이나 번역 과정 설명은 출력하지 말 것
