# 번역 지침

- 영어 원문을 한국어 독자가 처음부터 한국어로 쓴 글처럼 읽히게 번역한다.
- 핵심 주장, 논리 흐름, 뉘앙스, 예시는 보존한다.
- 영어식 어순과 직역투를 피한다.
- 문체는 담백하고 명확한 에세이체. 반말 금지, 과한 인터넷 말투 금지.
- `you`는 대부분 생략하거나 문맥에 따라 `우리는`, `사람들은`, `스스로`로 처리한다.
- 제목과 소제목은 자연스러운 한국어로 번역한다.
- Markdown 구조, 링크, 이미지, 리스트, 인용, 강조는 유지한다.
- YAML frontmatter는 유지하되 `title`, `subtitle`은 한국어로 번역하고 `original_title`, `original_subtitle`을 추가한다.
- 원문에 없는 내용은 추가하지 않는다.
